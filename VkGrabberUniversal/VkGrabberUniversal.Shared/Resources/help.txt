Основные настройки

Перед началом работы с приложением необходимо выполнить ряд настроек на главной странице. В первую очередь нужно заполнить список групп, в которых будет выполняться поиск записей по заданным критериям. Затем необходимо указать целевую группу, в которую будут добавляться выбранные записи.

Группы

Каждая группа имеет следующие поля:
- Активна - отображаются ли записи из этой группы в списке;
- Название или Id группы - короткое название группы или Id, которые можно найти в адресной строке браузера (https://vk.com/{название} или https://vk.com/public{id});
- Количество лайков - минимальное количество лайков, которое должна иметь запись, чтобы попасть в выдачу;
- Количество репостов - минимальное количество репостов, которое должна иметь запись, чтобы попасть в выдачу;
- Сдвиг - количество записей, которые нужно пропустить, начиная с самой свежей записи. Например, значение 230 ознает, что первые 230 записей будут пропущены и выдача начнется с 231 по порядку следования на стене записи.  При нажатии на кнопку "Загрузить еще" под списком постов значение сдвига автоматически увеличивается на 100.
Выдача записей формируется следующим образом: с сервера загружаются 100 записей, начиная со значения сдвига, и фильтруются по количеству лайков и репостов.

Планировщик

Позволяет добавлять записи в целевую группу по заданному расписанию. Для его работы необходимо заполнить следующие поля:
- "Начало" - время суток, начиная с которого будут добавляться записи;
- "Конец" - время суток, после которого записи добавляться не будут (планировщик перейдет к началу следующего дня);
- "Интервал" - период времени между записями;
- "Погрешность" (необязательное поле) - случайное время в диапазоне от нуля до указанного значения, которое будет добавляться или отниматься от интервала, чтобы избежать одинаковых промежутков между записями;
- "Дата добавления следующей записи" - дата, с которой начнется работа планировщика. При изменении этого значения планировщик продолжит работу с новой указанной даты.

Посты

Раздел содержит выборку постов по заданным критериям. Текст записей можно редактировать. Иллюстрации к записям увеличиваются при нажатии на них. Под увеличенным изображением расположена кнопка "Найти картинку", которая запускает браузер и выполняет поиск картинки в интернете. Под каждой записью отображается количество лайков, репостов и кнопка "Меню". С помощью меню запись можно:
- мгновенно добавить в целевую группу (кнопка "Добавить");
- добавить в отложенные записи с указанной датой публикации (кнопка "Добавить по дате");
- добавить запись с помощью планировщика, который сам высчитает дату публикации на основе предыдущей добавленной записи и заданных настроек (кнопка "В планировщик");
- открыть оригинал записи в браузере (кнопка "Открыть оригинал").
